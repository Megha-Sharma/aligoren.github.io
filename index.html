Windows'a Ruby on Rails Kurulumu

2015-04-18 12:41

Comments

Ruby on Rails yani RoR, Ruby'nin en gelişmiş web frameworklerinden biridir. Bu yazı yazıldığı vakitlerde en güncel sürümü 4.2'dir. Sitesine bu linkten ulaşabilirsiniz.

Windows'a Ruby kurulabilir ancak Rails gibi frameworklerin kurulumunda sıkıntı yaşanmakta. Bunun için RailsInstaller sitesi işi kolaylaştırıcı bir kurulum aracı hazırlamış. Şu an Windows ve Mac Os X için kurulum dosyaları bulunmakta.

RailsIntaller'a bu adresten gidebilirsiniz: http://railsinstaller.org/en

Aşağıda kendinize uygun Ruby sürümünü seçip indirme yapabilirsiniz.

RailInstaller gerekli olan bütün eklentileri içinde barındırıyor. Örneğin:

DevKit, Bundler, Sqlite, Git, TinyTDS ve SQL Server Support özellikleri.

Tabii ki Mac için var olan özellikler daha fazla.

Eğer daha detaylı bir kurulum yardımına ihtiyacınız varsa InstallRails sitesine gidebilirsiniz. START NOW butonuna tıklayarak size ait işletim sistemini seçip, kurulum dökümanlarına ulaşabilirsiniz.

Örneğin Windows bu linkten başlıyor: http://installrails.com/steps/railsinstaller_windows

Kurulum sonrasında bundle işleminde hata alanlar buraya bakabilirler: Ruby Bundle Install Error

Örneğin Ruby on Rails ile bir proje geliştiriyorsunuz. Bundle ile install diyerek gereksinimleri kurmak isteyeceksiniz. Ama o da ne bir hata aldınız:

C:\RailsWork\> bundle install

Gem::RemoteFetcher::FetchError: SSL_connect returned=1 errno=0 state=SSLv3 
read server certificate B: certificate verify failed 
(https://rubygems.org/gems/FALANCA.gem)
An error occured while installing FALANCA (FALAN_VERSIYON), and Bundler cannot continue.
Make sure that `gem install FALANCA -v 'FALAN_VERSIYON'` succeeds before bundling.

Bu hatayı geçmenizin bir yolu verilen gem linkini direkt olarak kopyalayıp

gem install FALANCA.gem

komutu ile kurmak. Ancak Rails gibi büyük frameworkleri kullanırken bağımlılıklar da çok fazla oluyorlar. Bu duruma kalıcı ve kesin bir çözüm getirmek gerekir.
