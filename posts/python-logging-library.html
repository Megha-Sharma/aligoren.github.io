Installation:

python setup.py install

Usage:

Using is simple. There is a sample code below

from handan import LogSaver
from handan import logMachine

@logMachine
def test(a,b):
    return a*b

log = LogSaver()

print(log.warn('This is tests the WARN logging call'))

You can also look test.py file.
