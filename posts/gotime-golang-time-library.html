GoTime Golang Time Library

2015-05-14 13:10

Comments

Go dilinde yeni olduğumu biliyorsunuz. Dili öğrenirken dilin imkanları ile daha önce istediğim ama fırsatımın bir şeyi yapmayı düşündüm. Bir zaman kütüphanesi.

Tam bir kütüphane sayılmaz aslında. Çünkü bir kütüphanenin sağladığı tüm özellikleri sağlıyor mu derseniz hayır sağlamıyor. Tanıtıma başlamadan önce kütüphaneye şuradan bakabilirsiniz:

Kütüphane Linki: https://github.com/aligoren/gotime

Gotime GoDoc Linki: https://godoc.org/github.com/aligoren/gotime

GoTime, bugünü, dünü, yarını, saat, zaman gibi kavramları gösterdiği gibi belirli zaman aralığında bekleme işlemini de yapıyor. Bu bekleme işlemi 6 farklı şekilde isteğinize göre gerçekleşebilir.

Nasıl Kullanılır?

Kütüphane linkine bakarsanız orada bir test klasörü bulunmakta o klasördeki main.go dosyasına bakarak kullanımını görebilirsiniz. Ancak yine de anlatmakta fayda var. Öncelikle Gotime'ı import edelim:

import "github.com/aligoren/gotime"

Eğer GOPATH ayarlarınız yapılandırılmış ise bu andan sonra kütüphaneyi kullanmaya başlayabilirsiniz. Biz öncelikle kütüphanedeki type function dediğimiz yapı fonksiyonlarına bakalım. Hangi fonksiyon ne iş görüyormuş görelim:

Zaman Fonksiyonları

Bu fonksiyonlar tarih ile ilgili değerleri verirler. Ay, Gün isimleri, zamanlar vs. gibi.

TimeNow() => Şu anki zamanı verir. Örneğin: 13:21:39
DayNow() => Bulunduğunuz günü sayısal olarak verir. Örneğin: 14
WeekNow() => Haftanın günü şeklinde metinsel bir değer verir. Örneğin: Perşembe
PreviousDay() => Bulunduğunuz günden bir önceki günü verir. Örneğin: Çarşamba
NextDay() => Yarını gösterir. Örneğin: Cuma
DayAgo() => X gün kadar öncesini söyler. Örneğin: 3 gün önce Pazartesi
DayLater() => X gün kadar sonrasını söyler. Örneğin: 3 gün sonra Pazar
MonthNow() => Şu anki ayı verir. Örneğin: May
MonthAgo() => X ay öncesi. Örneğin: 2 ay önce March
MonthLater() => X ay sonrası. Örneğin: 2 ay sonra July
YearNow() => Şu an bulunduğumuz yılı gösterir. Örneğin: 2015
YearAgo() => X yıl öncesini verir. Örneğin: 3 yıl önce 2012
YearLater() => X yıl sonrasını verir. Örneğin: 5 yıl sonra 2020
DayOfWeek() => Haftanın X. gününü metinsel olarak verir. Örneğin: 3. gün Çarşamba
DayOfYear() => Yılın kaçıncı gününde olduğunuzu söyler. Örneğin: 134. gün

Wait Fonksiyonlar

Bu fonksiyonlar belirlenen zamanlar için çalıştığınız projeyi uyku moduna alabilir. Wait fonksiyonları 6 tiptedir. Bu tipler:

WaitMicro() => X Mikrosaniye bekler
WaitNS() => X Nanosaniye bekler
WaitMS() => X Milisaniye bekler
WaitSecond() => X saniye kadar bekler
WaitMin() => X dakika kadar bekler
WaitHour() => X saat kadar bekler

Örnek Kullanım

package main

import (
    "fmt"
    "github.com/aligoren/gotime"
)

func main() {

    s := gotime.Now{}

    fmt.Println(s.TimeNow()) // Now Time 14:55:29 (My current time)
    fmt.Println(s.DayNow()) // Day Now (14th)
    fmt.Println(s.WeekNow()) // Weekday (Thursday)
    fmt.Println(s.PreviousDay()) // Wednesday May 13
    fmt.Println(s.NextDay()) // Friday May 15
    fmt.Println(s.DayAgo(3)) // 3 Days Ago: Monday
    fmt.Println(s.DayLater(2)) // 2 Days Later: Saturday
    fmt.Println(s.MonthNow()) // Now Month (May)
    fmt.Println(s.MonthAgo(3)) // 3 Months Ago: February
    fmt.Println(s.MonthLater(3)) // 3 Months Later: August
    fmt.Println(s.YearNow()) // Year Now: 2015
    fmt.Println(s.YearAgo(5)) // 5 Years Ago: 2010
    fmt.Println(s.YearLater(5)) // 5 Years Later: 2020
    fmt.Println(s.DayOfWeek(3)) // Day of Week 3th: Wednesday
    fmt.Println(s.DayOfYear()) // Day of Year 134th
    s.WaitSec(3) // Wait for 3 Seconds
    s.WaitMS(300) // Wait for 300 Milliseconds
    s.WaitMicro(5000) // Wait for 5000 Microsecond
    s.WaitNS(3000) // Wait for 3000 Nanoseconds
    s.WaitMin(3) // Wait for 3 Minutes
    s.WaitHour(5) // Wait for 5 Hours
}
