Gofret İndirme Yöneticisi Go Dili İle

2015-05-29 22:11

Comments

2 dakikada okunabilir.

Gofret GitHub: https://github.com/aligoren/gofret

Gofret Indir: https://github.com/aligoren/gofret/raw/master/build/gofret.exe

Gofret, go dilinde yazılmış olan wget benzeri, basit ve amatör bir indirme yöneticisidir. Program çoklu ortamda derlenebilir kaynak koda sahiptir. Bu bakımdan kullanımı rahattır.

Gofret için her ne kadar wget benzeri demiş olsam da onun yanından geçemeyeceği çok belli. Henüz go dilinde bellek yönetimi konusunda bir bilgiye sahip olmadığım için programın wget'e göre çok çok yavaş çalışması konusunda bir şey yapamadım.

Program GitHub üzerinde şu anlık sadece Windows için yayınlanmış durumda. Yukarıda zaten indirme linkini de paylaştım.

Neden Gofret adını seçtiğime gelince kodları yazarken canım gofret çekti o ana kadar build ettiğim dosyaya ne ad vereceğimi dahi bilmiyordum. Şu yazıyı yazdığım sıralarda canım hala gofret çekiyor.

Teknik bir özellik sunmayı çok isterdim. Ancak sunacağım tek teknik özellik ise yardımcı komutlar olacaktır sanırım. Bu komutlar:

-version: Program sürümünü belirtir.

-help: Yardım gösterir. Bu kısım henüz boş gibi sayılır.

-about: Hakkında kısmı burada kendimden bahsettim galiba :)

Eğer GitHub üzerindeki repo'yu kullanıp kendiniz build etmek isterseniz öncelikle Sergey Cherepanov adlı arkadaşın pb kütüphanesini kurmanız gerekmekte.

pb kütüphanesi kurulumu:

go get github.com/cheggaaa/pb

Kurulumdan sonra ise build etme işlemine geliyoruz. Build etme işlemi çok zor değil. Sadece platform için tek bi farklılık var.

go build -o gofret main.go // linux ve unix benzeri sistemler için
go build -o gofret.exe main.go // windows için

Hangi platformu kullanıyorsanız ona göre derleme işlemini yapabilirsiniz.

Kullanım

Derleme işlemini yaptıktan sonra kullanmak istediğinizde şunu yapabilirsiniz:

./gofret -url=https://site.com/dosya.zip

Bu kullanım ile dosya indirme işlemi başlayacaktır. Örnek görsel de vereyim.
